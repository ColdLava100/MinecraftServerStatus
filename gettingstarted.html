Getting Started

Welcome to Lokal SMP!

1. Connecting to the Server

Open Minecraft and go to Multiplayer. Click "Add Server" and enter the following details:

Java Edition Server Address: lokalsmp.ddns.net:59490

Bedrock Edition Address: lokalsmp.ddns.net:58942

2. Server Rules

  • Be respectful to others.
  • No griefing or stealing.
  • No hacking or exploiting bugs.
  • Keep chat friendly and appropriate.

3. Useful Commands

  • /claim - Used for claiming Land.
  • /waypoints - Used for making waypoints.
